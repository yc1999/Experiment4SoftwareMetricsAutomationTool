# Experiment4SoftwareMetricsAutomationTool
此项目以中南大学杨柳老师讲解的中国大学MOOC[《软件度量及应用》](https://www.icourse163.org/learn/CSU-1001907005?tid=1450755801#/learn/announce)为理论基础，通过使用Java Web技术实现了以下6项自动化软件度量功能：
- 需求分析阶段的度量
  - 功能点度量
  - 用例点度量
- 详细设计阶段的度量
  - 圈复杂度度量
  - 面向对象LK度量
  - 面向对象CK度量
- 编码实现阶段的度量
  - 代码行度量

祝您使用愉快！
